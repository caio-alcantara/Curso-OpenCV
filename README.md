# Curso OpenCV

Este repositório contém materiais e exemplos práticos para aprendizado de OpenCV, uma biblioteca poderosa para processamento de imagens e visão computacional. A seguir está um resumo das principais pastas e seus conteúdos:

## Estrutura do Repositório

1. **[Faces](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/Faces)**
   - Scripts para detecção e manipulação de rostos usando OpenCV.

2. **[Photos](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/Photos)**
   - Exemplos e arquivos de imagens usados nos exercícios.

3. **[Videos](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/Videos)**
   - Exemplos de vídeos utilizados para demonstração de manipulação e processamento.

4. **[basic_functions](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/basic_functions)**
   - Implementações de funções básicas como leitura, escrita e exibição de imagens.

5. **[bitwise_operations](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/bitwise_operations)**
   - Operações bitwise (AND, OR, XOR, NOT) para manipulação de pixels.

6. **[blurring](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/blurring)**
   - Técnicas de suavização de imagens, como blur e GaussianBlur.

7. **[color_boundaries](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/color_boundaries)**
   - Detecção de limites de cores em imagens.

8. **[color_channels](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/color_channels)**
   - Manipulação de canais de cor (RGB, HSV, etc.).

9. **[color_detection](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/color_detection)**
   - Scripts para identificação de cores específicas.

10. **[color_spaces](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/color_spaces)**
    - Conversão entre diferentes espaços de cor (RGB, HSV, LAB, etc.).

11. **[contours](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/contours)**
    - Extração e análise de contornos em imagens.

12. **[drawing_and_text](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/drawing_and_text)**
    - Desenho de formas geométricas e adição de textos nas imagens.

13. **[edge_detection](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/edge_detection)**
    - Detecção de bordas usando algoritmos como Canny e Sobel.

14. **[histograms](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/histograms)**
    - Análise de histograma de imagens para distribuição de intensidade de cores.

15. **[image_transformation](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/image_transformation)**
    - Transformações geométricas, como rotação, redimensionamento e translação.

16. **[masking](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/masking)**
    - Criação de máscaras para processar regiões específicas de uma imagem.

17. **[reading_images_and_video](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/reading_images_and_video)**
    - Exemplos de leitura e manipulação de imagens e vídeos.

18. **[repo_base](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/repo_base)**
    - Scripts base para organização do repositório.

19. **[tresholding](https://github.com/caio-alcantara/Curso-OpenCV/tree/main/tresholding)**
    - Técnicas de thresholding para segmentação de imagens.
